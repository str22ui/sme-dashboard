# Dashboard NPL - Production

TV Dashboard untuk monitoring NPL per kanwil dan cabang.

## Deployment ke Vercel

1. Push ke GitHub
2. Import ke Vercel
3. Set Root Directory: `dashboard-npl`
4. Deploy!

## Features

- Auto-refresh setiap 30 detik
- Keyboard navigation (arrow keys)
- Auto-slide dengan pause (Enter key)
- Line charts untuk trend analysis
- Timezone: Asia/Jakarta (WIB)

## Data Source

Dashboard reads from Vercel Blob storage yang diupload via Admin portal.
